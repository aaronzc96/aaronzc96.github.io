​ else if和else不可以另起一行

  • switch

  1. 表达式val1和val2类型必须相同

  2. 不需要添加break

  3. 要想在进入一个case后继续执行后面的case需要加fallthrough,可以无条件进入下一个case, 如果需要无条件进入下面几个case需要在每个case中加fallthrough

  • select
  1. 每个case都是一个通信
  2. 每个channel表达式都会被求值
  3. 所有被发送的表达式都会被求值
  4. 只执行一个可以进行的通信，同时有几个可以公平地选育各执行
  5. 没有可以执行的，有default则执行，没有则阻塞

切片

切片是对被切数组的部分引用

  1. len(slice_name)获取长度，cap(slice_name)获取容量
  2. 未初始化时切片为nil
  3. append(slice_name, element)向切片最后加入元素（细节的坑比较多，可以自行研究）
  4. copy(dst, src)复制切片

并发

关键字go开启goroutine

  • go开启运行期线程
  • 同一程序goroutine共享地址空间

通道channel用来传递数据，用于goroutine之间进行同步和通讯<-指定通道的方向

带缓冲区的channel

  • 不带缓冲区：发送方会阻塞直到接收方接收
  • 带缓冲区：
    • 接收方无值接收会一直阻塞
    • 发送方阻塞直到发送数据拷贝进缓冲区
    • 缓冲区满了则发送方需要等接收方接收到一个值

可以使用range遍历通道中的值，用close关闭通道
